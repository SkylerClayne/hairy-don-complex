Hair Don Complex

Here I've embedded a few static pages, these pages are projects which I had put together for a friend (my residence don, hence the repo name). He was taking an introduction to web development course for non-computer science majors in 2014, with a very ancient professor. With my kind heart, I used these examples the professor gave to teach him the ways of a front end web developer! They're really nothing special, and shouldn't be taken in anyway as a broadcast of my skill but as my willingness to help and teach.

Course Selection

This is the first and last assignment my don was given. First we had to implement a skeleten, with no functionality only the looks and feels. I know, I know, this is a terrible design, but, this is what you get when you give your students a terrible design to exactly copy. I do wish in this course they taught students some web frameworks like maybe bootstrap...

JavaScript Exercise

This was the part of the course where they introduced javascript, yay actualy programming -- sort of... We had to take in a string and format it in a bunch of ways. This would have been a million times easier and effective with jQuery.

Conclusions

Although this project was a very large toy, it was a fun reminder as to why we take jQuery and bootstrap for granted. An idea that I think might be really fun is to re-create these using bootstrap and jQuery to show how much mroe power these tools give us.
All in all, I really enjoyed helping my don. We gained an awesome relationship. Often we have wings and drinks together (I have beer, he has a regular coke).
